# KPU_ComDesign
한국공학대학교 AR Navigation System + Info System

# 프로젝트 기획 의도
신입생들과 편입생들 그리고 재학중인 학생들을 위한 공학관 별 Navigation 시스템 구현 및 코로나 시국으로 인한 출입 제한으로 인해 학교에 대해 궁금증을 가질 수 있는 학부모를 위한 어플리케이션 개발
